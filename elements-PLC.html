Rockwell Micro850 PLC and Ignition HMI/SCADA for Water Level Control

RSLogix 5000 for PLC based control of water level. Cogent Datahub SCADA setup and Ignition software HMI design.

PLC based Water level control in practice

Work

  • Established connection to Rockwell's Connected Components Workbench (CCW) Micro850 PLC Simulator over ethernet IP.
  • Programmed the PLC in RSLogix 5000 ladder logic, structured text, and function blocks.
  • Designed Human Machine Interface (HMI) with Ignition Designer Software and setup an Open Platform Communications (OPC) client.
  • Developed SCADA system with OPC DA and UA server-client setup using Cogent Datahub.
  • Configured OPC DA client to communicate with Micro850 PLC, and OPC UA server to communicate with Ignition HMI.
